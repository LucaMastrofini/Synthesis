This VST plugin has been developed for **academic purposes**. It is intended as a tool for learning and experimentation with audio synthesis techniques. 

The plugin is **open source**, and the source code is available on GitHub:

## Installation
1. Download the latest release from the [GitHub Repository](https://github.com/LucaMastrofini/Synthesis).
2. Copy the `.vst` file to your VST plugin directory.
   - On Windows: `C:\Program Files\VSTPlugins`
   - On macOS: `/Library/Audio/Plug-Ins/VST`
3. Launch your DAW (Digital Audio Workstation) and scan for new plugins.

## Usage
- Load the plugin into your DAW.
- Experiment with the provided controls to explore its synthesis capabilities.
